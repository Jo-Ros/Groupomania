Groupomania

Réseau Social d'entreprise

Prérequis:
  Node /
  Angular

Clone:
git clone https://github.com/Jo-Ros/Groupomania.git

Backend:
    Technologies utilisées:
      NodeJs /
      Express /
      MongoDB

    Installation:
      cd back-end;
      npm install;
      Créez un dossier image;
      nodemon server;
  
 Frontend:
    Technologies utilisées:
    Angular /
    TypeScript /

    Installation:
      cd front-end;
      npm install;
      ng serve ( ng serve --o pour ouvrir directement dans votre navigateur par défaut );
